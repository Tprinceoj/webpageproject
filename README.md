# WebPageProject

## Project Overview
This simple web project consists of a single HTML page (`index.html`), a stylesheet (`styles.css`), and a JavaScript file (`scripts.js`). The page showcases my favorite hobby—painting—with a heading, descriptive paragraph, an image, and a link to further resources. A basic stylesheet adds structure and visual polish.

## My Experience
Working on this project was a great introduction to building a static web page from scratch. I created the HTML structure, linked external CSS and JavaScript, and styled elements to improve the user experience. Handling the file organization and linking resources helped solidify how web assets interact.

## Challenges Faced
One challenge was ensuring the CSS file was properly linked and the styles applied correctly; an empty stylesheet initially left the page unstyled until I added the rules. Another small hiccup involved moving content between files and deleting duplicates without breaking links.

## What I Learned
- How to structure a basic HTML document with `<!DOCTYPE>`, `head`, and `body` elements.
- Linking external CSS and JavaScript files to an HTML page.
- Applying basic styles for typography, images, and links using CSS.
- Organizing project files and cleaning up unnecessary ones (e.g. deleting `newpage.html`).

This project laid the foundation for more advanced web development tasks and gave me confidence in creating and styling web pages.
